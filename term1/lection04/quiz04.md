## Вопрос 2

**Чем файл `.cf` отличается от `.dt` по составу и типичным условиям выгрузки?**

1. `.cf` всегда включает все данные ИБ  
2. `.cf` и `.dt` по составу не отличаются  
3. `.cf` можно получить только через `pg_dump`  
4. `.cf` содержит только конфигурацию; часто выгружают при работающих пользователях

## Вопрос 4

**Почему при изменении длины строкового реквизита (реструктуризация) часто требуется монопольный доступ?**

1. Потому что платформа не умеет менять строки  
2. Потому что нужна реструктуризация таблиц в СУБД без параллельной работы пользователей

4. Потому что `.cf` нельзя сохранить иначе

## Вопрос 7

**Чем по смыслу семинара отличаются `ТекущаяДатаСеанса` и `ТекущаяДата()`?**

1. `ТекущаяДата()` используется только в файловой базе  
2. Они всегда равны независимо от настроек  
3. Обе функции возвращают время только в UTC  
4. `ТекущаяДатаСеанса` — логическая дата пользователя (часовые пояса); `ТекущаяДата()` — дата сервера

## Вопрос 9

**Зачем нужны блокировки по измерениям (пример: склад + номенклатура)?**

1. Чтобы запретить любые продажи  
2. Блокировки в 1С не используются  
3. Чтобы ускорить запрос  
4. Чтобы два пользователя не «разобрали» один остаток параллельно и не получили отрицательный остаток
