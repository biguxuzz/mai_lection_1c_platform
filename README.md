# Курс "Разработка на платформе 1С:Предприятие"

## Общая информация

Двухсеместровый курс для студентов Факультета информационных технологий, охватывающий основы разработки на платформе 1С:Предприятие и современные подходы к разработке с использованием искусственного интеллекта.

**Первый семестр:** 2 курс — основы платформы 1С:Предприятие  
**Второй семестр:** 3 курс — разработка на платформе 1С с использованием ИИ

---

## Первый семестр: Основы разработки на платформе 1С:Предприятие

### Цель курса
Освоить базовые концепции и инструменты разработки на платформе 1С:Предприятие, изучить объектную модель данных, научиться создавать конфигурации и реализовывать бизнес-логику.

### Программа лекций (10 лекций)

1. **[Введение в платформу 1С:Предприятие](term1/lection1/lection1.md)**
   - История развития платформы (версии 7 и 8)
   - Архитектура и области применения
   - Клиент-серверная архитектура
   - Процесс внедрения и роли на проекте

2. **[Объекты хранения данных](term1/lection2/lection2.md)**
   - Константы, справочники, документы
   - Регистры накопления, сведений и бухгалтерии
   - Планы видов характеристик
   - Итоги и виртуальные таблицы

3. **[Роли и полномочия](term1/lection3/lection3.md)**
   - Система ролей и пользователей
   - Права доступа (интерактивные и неинтерактивные)
   - Права на уровне записи (RLS)
   - Отладка и замер производительности

4. **[Работа с формами и параметрами сеанса](term1/lection4/lection4.md)**
   - Проектные технологии разработки
   - Инструмент разработки EDT
   - Блокировки данных
   - Параметры сеанса

5. **[Базовые типы данных и коллекции](term1/lection5/lection5.md)**
   - Примитивные и составные типы данных
   - Коллекции значений (массивы, структуры, таблицы значений)
   - Объектная модель работы с данными
   - Асинхронное программирование

6. **[Поставка конфигурации и расширения](term1/lection6/lection6.md)**
   - Настройка конфигурации для поставки
   - Обновление конфигурации
   - Расширения конфигурации
   - Правила работы с расширениями

7. **[Веб-сервисы и HTTP-сервисы](term1/lection7/lection7.md)**
   - Создание веб-сервисов
   - Публикация на веб-сервере
   - Создание клиентов для веб-сервисов
   - Права доступа и привилегированный режим

8. **[Транзакции и блокировки](term1/lection8/lection8.md)**
   - Неявные и явные транзакции
   - Типы блокировок (разделенная, эксклюзивная)
   - Контроль остатков
   - Управляемые блокировки данных

9. **[Работа с хранилищем](term1/lection9/lection9.md)**
   - Временное хранилище для передачи данных
   - Хранилище значений для картинок
   - Асинхронные вызовы с файлами

10. **[Язык запросов в 1С:Предприятие](term1/lection10/lection10.md)**
    - Основы синтаксиса языка запросов
    - Конструктор запросов
    - Соединения таблиц и виртуальные таблицы
    - Временные таблицы и подзапросы
    - Оптимизация запросов

### Практические занятия (семинары 4-10)

- **[Семинар 4:](term1/seminar4/seminar4.md)** Печатные формы, ввод на основании, задачи и бизнес-процессы
- **[Семинар 5:](term1/seminar5/seminar5.md)** Инструменты групповой разработки (хранилище 1С, EDT, Git, Cursor)

- **[Семинар 9:](term1/seminar9/seminar9.md)** Ревью конфигураций и рекомендации по проектированию

### Курсовая работа первого семестра

**Тема:** Разработка конфигурации 1С:Предприятие

- Версионирование конфигурации

## Второй семестр: Разработка на платформе 1С с использованием ИИ

### Цель курса
Освоить современные подходы к разработке с использованием искусственного интеллекта, изучить агентное программирование, RAG-системы, интеграцию LLM с платформой 1С и автоматизацию процессов разработки.

### Программа лекций (16 лекций)

1. **Введение в агентское программирование**
   - Что такое агентское программирование
   - Актуальность и применение в современной разработке
   - Обзор возможностей для платформы 1С

2. **LLM - история и инструменты**
   - Краткая история развития LLM
   - Проприетарные vs локальные модели
   - Инструмент LLM Studio - установка и настройка

3. **IDE и CLI для разработки с ИИ**
   - Обзор популярных IDE с поддержкой ИИ (Cursor, VSCode, 1C:EDT)
   - CLI-приложения для работы с LLM
   - Настройка рабочего окружения

4. **Промптинг и интеграция с API**
   - Основы промпт-инженерии
   - Техники сбора и подготовки контекста
   - Практическая разработка интеграции с API LLM

5-6. **Docker - основы и практика**
   - Контейнеризация и Docker
   - Основные концепции: образы, контейнеры, volumes
   - Docker Compose
   - Сборка приложений в Docker
   - Best practices

7. **[RAG (Retrieval-Augmented Generation) - теория](term2/lection6/)** *(материалы в папке)*
   - Что такое RAG и зачем он нужен
   - Архитектура RAG-систем
   - Векторные базы данных (ChromaDB, Qdrant, Milvus)
   - Embedding модели и chunking стратегии
   - **Графовые RAG системы (Graph RAG)**
   - **LightRAG - современная реализация Graph RAG**

8. **[RAG - практика и продвинутые техники](term2/lection8/lection8.md)**
   - Решение проблем векторного RAG
   - Демонстрация семантического поиска
   - GraphRAG и Neo4j
   - Практическая реализация LightRAG
   - Интеграция в n8n workflow

9-10. **n8n - введение и интеграция с 1С и ИИ**
   - Low-code автоматизация
   - Основные концепции: nodes, workflows, triggers
   - HTTP запросы к 1С через OData/HTTP-сервисы
   - Интеграция с OpenAI API и локальными LLM
   - Автоматизация документооборота с помощью ИИ

11-12. **MCP (Model Context Protocol) серверы**
   - Что такое Model Context Protocol
   - Архитектура MCP серверов
   - Развертывание MCP сервера
   - Создание собственных MCP tools для 1С
   - Интеграция MCP с EDT
   - Настройка доступа к метаданным конфигурации

13-14. **Разработка интерфейсов в 1С**
   - Типы форм в 1С
   - Использование ИИ для генерации кода форм
   - Промпты для создания типовых интерфейсов
   - Динамическое создание элементов форм
   - Рефакторинг существующих интерфейсов с помощью LLM

15. **Безопасность при разработке с использованием ИИ**
   - Риски при использовании ИИ в разработке
   - Утечка конфиденциальных данных
   - Безопасное использование публичных LLM API
   - Локальные модели vs облачные сервисы
   - Best practices для корпоративной разработки

16. **Агентные системы - архитектура и реализация**
   - Архитектура агентных систем (ReAct, Plan-and-Execute)
   - Создание агента с доступом к инструментам
   - Память и контекст агента
   - Multi-agent системы
   - Практика: создание агента для работы с 1С

### Практические занятия (4 занятия)

1. **[Настройка необходимого окружения](term2/seminar1/seminar1.md)**
   - Настройка плагина Continue в VS Code
   - Работа с локальными и удаленными моделями
   - Настройка MCP серверов (векторный и графовый RAG)
   - Создание кода модуля для конфигураций

2. **Автоматизация с n8n**
   - Создание workflow для обработки документов из 1С
   - Интеграция с LLM для анализа данных
   - Автоматическая генерация отчетов

3. **Разработка интерфейсов с помощью ИИ**
   - Генерация форм документов с помощью промптов
   - Автоматизация создания типовых элементов
   - Рефакторинг существующих форм

4. **Создание ИИ-агента для 1С**
   - Разработка агента с набором инструментов
   - Интеграция агента в конфигурацию
   - Тестирование и демонстрация работы
