# Семинар 2: Запуск простейшего агента для работы с базой данных 1С

**Продолжительность:** 2 пары по 90 минут (всего ~180 минут)  
**Целевая аудитория:** Студенты 3 курса, Факультет информационных технологий  
**Курс:** Разработка на платформе 1С с использованием ИИ

## Цель семинара
Настроить MCP-сервер в базе данных 1С, создать простейший инструмент для получения данных из базы, проверить его работу и интегрировать с n8n для создания ИИ-агента, который может получать отчеты из базы 1С по запросам пользователя.

## Необходимые предзнания
- Основы работы с платформой 1С:Предприятие
- Базовые знания о расширениях конфигураций 1С
- Понимание концепций MCP (из семинара 1)
- Базовые навыки работы с Docker
- Понимание работы HTTP-сервисов в 1С
- Базовые знания о n8n или других системах автоматизации

##### 5.4 Реализация метода ВыполнитьИнструмент (7 минут)

**Реализация логики получения данных:**
```bsl
Функция ВыполнитьИнструмент(ИмяИнструмента, Аргументы) Экспорт
    
    Если ИмяИнструмента = "get_my_data" Тогда
        
        ПериодНачало = Дата(Аргументы.period_start);
        ПериодКонец = ?(Аргументы.Свойство("period_end"), 
                         Дата(Аргументы.period_end), 
                         ТекущаяДата());
        
        Запрос = Новый Запрос;
        Запрос.Текст = 
        "ВЫБРАТЬ
        |    МоиДанные.Ссылка КАК Ссылка,
        |    МоиДанные.Наименование КАК Наименование
        |ИЗ
        |    Справочник.МоиДанные КАК МоиДанные
        |ГДЕ
        |    МоиДанные.ДатаСоздания МЕЖДУ &ПериодНачало И &ПериодКонец";
        
        Запрос.УстановитьПараметр("ПериодНачало", ПериодНачало);
        Запрос.УстановитьПараметр("ПериодКонец", ПериодКонец);
        
        РезультатЗапроса = Запрос.Выполнить();
        
        Данные = Новый Массив;
        ВыборкаДетальныеЗаписи = РезультатЗапроса.Выбрать();
        Пока ВыборкаДетальныеЗаписи.Следующий() Цикл
            
            Элемент = Новый Структура;
            Элемент.Вставить("reference", Строка(ВыборкаДетальныеЗаписи.Ссылка));
            Элемент.Вставить("name", ВыборкаДетальныеЗаписи.Наименование);
            Данные.Добавить(Элемент);
            
        КонецЦикла;
        
        Результат = Новый Структура;
        Результат.Вставить("content", Новый Массив);
        Результат.content.Добавить(Новый Структура("type", "text", "text", JSON.stringify(Данные)));
        
        Возврат Результат;
        
    КонецЕсли;
    
    ВызватьИсключение "Неизвестный инструмент: " + ИмяИнструмента;
    
КонецФункции
```

**Ключевые моменты:**
- Получение параметров из аргументов
- Выполнение запроса к базе данных
- Преобразование результатов в формат JSON
- Формирование ответа в формате MCP
- Обработка ошибок

##### 8.2 Интеграция ИИ-агента с MCP-сервером (7 минут)

**Создание полного workflow:**
1. Триггер: получение запроса от пользователя (Webhook или Chat Trigger)
2. ИИ-узел: анализ запроса пользователя и определение параметров
3. HTTP Request: вызов инструмента MCP-сервера с определенными параметрами
4. ИИ-узел: форматирование ответа для пользователя
5. Ответ пользователю

**Логика работы:**
- Пользователь задает вопрос на естественном языке
- ИИ-агент анализирует запрос и определяет, какие данные нужны
- Агент формирует запрос к MCP-серверу
- Полученные данные форматируются в понятный ответ
- Ответ отправляется пользователю

**Тестирование агента:**
- Запрос данных на естественном языке
- Проверка корректности определения параметров
- Проверка получения данных
- Проверка качества ответа

**Результат:** ИИ-агент создан и интегрирован с MCP-сервером 1С.
